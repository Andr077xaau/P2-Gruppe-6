 Breaking it down:

spring.datasource.url=jdbc:mysql://localhost:3306/energi_forbrug?serverTimezone=UTC
  jdbc:mysql://  localhost  :  3306  /  energi_forbrug  ?serverTimezone=UTC
  ↑              ↑              ↑        ↑                ↑
  protocol    your PC       MySQL     database name     fixes time-related
                          default port                  errors in MySQL

  ---
  spring.datasource.username=energi_app
  spring.datasource.username  =  energi_app
  ↑                               ↑
  tells Spring this is            the MySQL user you
  the DB username                 created in your SQL script

  ---
  spring.datasource.password=YOUR_PASSWORD_HERE
  spring.datasource.password  =  YOUR_PASSWORD_HERE
  ↑                               ↑
  tells Spring this is            replace this with the actual
  the DB password                 password from your SQL script

  ---
  spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
  spring.datasource.driver-class-name  =  com.mysql.cj.jdbc.Driver
  ↑                                        ↑
  which JDBC driver to use                 the MySQL driver class
                                           from mysql-connector-j

  ---
  spring.jpa.hibernate.ddl-auto=validate
  spring.jpa.hibernate.ddl-auto  =  validate
  ↑                                  ↑
  what to do with DB tables           only CHECK that tables match
  on app startup                      your code — don't change anything

  ---
  spring.jpa.show-sql=true
  spring.jpa.show-sql  =  true
  ↑                        ↑
  print SQL queries         yes, show them
  to console                in the terminal

  ---
  server.port=8080
  server.port  =  8080
  ↑                ↑
  which port       your app runs at
  to use           http://localhost:8080
